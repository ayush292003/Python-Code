# BITCOIN PRICE TRACKER

- CLI Based Tool to fetch real time bitcoin price data
- Supports Over 100 Currencies


### COMMANDS

- -btc <CURRENCY_SYMBOL>
    returns real time price of bitcoin in that currency
- -w <FILENAME.json>
    to save the fetched data into a json file
